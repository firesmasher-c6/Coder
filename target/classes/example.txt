# This is Coder Plugin
# Coder Plugin Introduces Phyton And Java execution
# This is example.txt

# For Phyton Scripts:
Coder only reads .py and .java files,
Those .py files should be in Coder/Phyton/scripts/

# For Java Scripts:
Coder reads .java files inside Coder/Java/scripts/
these are executed manually through commands,

# Commands:
/coder java run <name> - runs the java file 
/coder phyton run <name> - runs the phyton file 
/coder scripts reload - reloads the entire Java and Phyton Folder
/coder phyton reload <scriptname> - reloads the specific .py file
/coder java reload <scriptname>  - reloads the specific .java file
