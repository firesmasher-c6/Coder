# Coder User Guide

This guide explains how to use the three scripting types supported by Coder and how to manage them via commands.
## File Extensions
You can only run **Phyton, Java, Lua** codes.
The file extensions will be:
**.py** for Phyton,
**.java** for Java,
**.lua** for Lua.

## How Does Coder Work?
Coder Runs on the JVM (Java Virtual Machine).
this uses **Jythton** for Phyton,
and **LuaJ** for Lua.
The Java is based on your hardware, if you run Java 25 you can run Java 25 codes.

## 1.3.6 Includes in the future:
**an html file that you can download and made by the server when you do /coder start website**
**that html file lets you run coder commands on the browser, see console error logs and more.**
**this is safe as this html file is generated by the machine.**

## NOTE
**Jython** is stuck on Phyton 2.7, you **cannot** run Phyton **3** codes.
**LuaJ** is at Lua 5.3, you **cannot** run Lua **5.4 or newer** codes.
